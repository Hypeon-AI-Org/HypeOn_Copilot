# HypeOn Copilot Frontend

AI-powered e-commerce intelligence platform frontend built with Next.js.

## Quick Start

### Installation

```bash
npm install
```

### Environment Variables

Create `.env.local`:

```env
NEXT_PUBLIC_API_BASE_URL=https://api.copilot.hypeon.ai
```

### Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
npm start
```

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Import in [Vercel](https://vercel.com)
3. Add `NEXT_PUBLIC_API_BASE_URL` environment variable
4. Deploy

### Other Platforms

Build the application and deploy the `.next` output directory according to your platform's Next.js deployment instructions.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
